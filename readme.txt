# Deskripsi:
News API App adalah aplikasi web sederhana yang memungkinkan pengguna mencari berita berdasarkan kata kunci atau kategori.  Aplikasi ini menggunakan Flask sebagai backend dan News API untuk mendapatkan data berita secara real-time.

# Fitur:
- Pengguna dapat mencari berita berdasarkan kata kunci atau kategori tertentu seperti Business, Technology, Sports, Health, Entertainment, dan Science.
- Aplikasi menampilkan hasil pencarian berita dalam bentuk daftar artikel.
- Tampilan berita dengan judul, deskripsi, dan tautan ke sumber berita asli.

# Teknologi yang Digunakan:
- HTML5 untuk struktur dan tampilan aplikasi.
- CSS3 untuk desain dan styling aplikasi.
- Flask sebagai backend untuk memuat daftar berita dan menangani pencarian berita dari News API.
- News API untuk mendapatkan daftar berita terkini, sumber: https://newsapi.org/.

# Cara Penggunaan:
- Menjalankan backend (app.py) dengan menggunakan perintah `python app.py` di terminal.
- Mengakses aplikasi web di browser dengan alamat `http://localhost:5000/ (sesuai arahan Flask).
- Menggunakan form pencarian untuk memasukkan kata kunci atau kategori berita yang diinginkan.
- Menampilkan hasil pencarian berita dalam bentuk daftar artikel.
- Mengklik judul artikel untuk melihat detail berita dan tautan ke sumber asli berita.
